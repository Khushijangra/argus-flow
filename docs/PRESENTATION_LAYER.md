These components execute successfully in the browser, but are primarily cosmetic or read-only consumers.

- **Digital Twin**: Renders canvas animations. (Cosmetic; not perfectly bound to RL queue data).

- **Scenario Studio**: Triggers the mocked `/api/inject` endpoint.
- **Network Grid**: Displays mock data for junctions J1-J4, J6-J16, and actual data for J5.
